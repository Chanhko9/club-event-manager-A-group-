# club-event-manager-A-group-

\# Club Event Manager



Dự án web quản lý sự kiện câu lạc bộ.



\## Công nghệ sử dụng

\- Frontend: HTML, CSS, JavaScript

\- Backend: Node.js

\- Database: MySQL



\## Cấu trúc thư mục

\- `frontend/`: giao diện người dùng

\- `backend/`: server và API

\- `database/`: file SQL tạo bảng và dữ liệu mẫu
